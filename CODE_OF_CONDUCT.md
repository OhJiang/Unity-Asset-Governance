# Code of Conduct

## Our Standard

Project discussions and contributions must remain respectful, constructive, and focused on helping
people improve the software. Welcome questions from contributors with different levels of Unity and
C# experience.

Acceptable participation includes:

- Giving specific technical feedback without personal attacks.
- Asking for clarification before assuming intent.
- Respecting different backgrounds, identities, and experience levels.
- Acknowledging mistakes and correcting them openly.

Harassment, discrimination, threats, deliberate intimidation, sexualized content, or repeated
hostile behavior are not acceptable.

## Enforcement

Repository maintainers may edit or remove comments, issues, pull requests, or other contributions
that violate this standard. Repeated or severe violations may result in temporary or permanent
exclusion from project spaces.

Report conduct concerns privately to the repository maintainer through the contact methods available
on the maintainer's GitHub profile. Reports should include enough context for a fair review and will
be handled as confidentially as practical.
